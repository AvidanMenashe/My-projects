This code implements a-binary classifier by Logistic Regression model.
The model is classified digits one and two as 0 and 1 respectively.
The data set I used in order to train and test the model is mnist.
The model has a success rate of 93.75%
You must make sure that the mat file- mnist_all, appers in the same directory of the code in order to sucessed load the data to the code file.
